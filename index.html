API 金鑰
金鑰僅存於您的瀏覽器本地（localStorage），不會傳送至任何第三方。
⚠ API 無法連線
用 file:// 直接開啟 HTML 時，瀏覽器安全機制會封鎖 API 請求。
以下任一方式可解決，選擇最方便的：
1
Python（最推薦・幾乎所有電腦都有）
在 HTML 檔所在資料夾開終端機，執行：
python -m http.server 8765
然後瀏覽器開啟 http://localhost:8765，點選 HTML 檔名。
Mac/Linux 可能需要用 python3
2
Node.js / npx
npx serve . -p 8765
開啟 http://localhost:8765 → 點 HTML 檔名。
3
VS Code → Live Server 插件
在 VS Code 開啟 HTML → 右下角點 Go Live，自動啟動本地伺服器。
💡 啟動伺服器後，原本這個 file:// 視窗就不需要了。
改從 http://localhost:8765 開啟，API 就能正常使用。
