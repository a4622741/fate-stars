API 金鑰

⚠ API 無法連線
用 file:// 直接開啟 HTML 時，瀏覽器安全機制會封鎖 API 請求。
以下任一方式可解決，選擇最方便的：
1
Python（最推薦）
在資料夾開終端機，執行：
python3 -m http.server 8765
然後瀏覽器開啟 http://localhost:8765
2
Node.js / npx
npx serve . -p 8765
開啟 http://localhost:8765
3
VS Code → Live Server
右下角點 Go Live。
💡 改從 http://localhost:8765 開啟，API 就能正常使用。
